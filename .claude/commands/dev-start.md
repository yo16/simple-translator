# 開発フェーズ開始

以下の手順を実行してください。

## ステップ1: 実行可能タスクの取得
Agent(`beads-manager`) に `bd ready --json` を実行させ、依存関係が解消されたタスク一覧を取得する。

## ステップ2: タスクの実行
取得したタスクそれぞれについて `/dev-task {id}` の手順を実行する。

### 並列実行ルール
- 依存関係のないタスク同士は、同一メッセージ内で複数のAgent呼び出しにより並列実行可能
- 並列実行するタスクは `isolation: worktree` のエージェントを使う
- 同じファイルを編集する可能性があるタスクは順次実行に切り替える

## ステップ3: 繰り返し
タスク完了後、再度 `bd ready` で次の実行可能タスクを取得し、すべてのタスクが完了するまで繰り返す。

## ステップ4: 全タスク完了
すべてのタスクが完了したら、ユーザーに開発フェーズ完了を報告する。
